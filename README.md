# [Crust & Crumb](https://crustandcrumb.onrender.com/)

E-commerce application for a fictional bakery client, featuring MVC architecutre.

Demonstrates:
- Database Design
- Routing
- Customer Relationship Manager (CRM)
- E-commerce
- Styling
